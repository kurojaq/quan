# Rules

**The Three-Body Stability Protocol** — Self, Market, and Framework must
all be in a stable configuration simultaneously (the "Lagrange
configuration") before the Mission Brief opens:

| Body | Stable | Unstable |
|---|---|---|
| **Self** | Internal clock entrained to CW arc; hunter states below threshold; clean Strategist/Ground-Lead switching; Psyq log coherence>0.80 | Clock inverted/drifted; switching failing; unresolved emotional residue |
| **Market** | Low entropy consumption, strong directional prior, clean-gradient PDSLs, Jerk<10, ℛ_F<2.0 | ZC>4, conflicted prior, ambiguous gradient, high Jerk, ℛ_F near veto |
| **Framework** | Applied completely and in sequence, no unresolved gaps, AAR findings producing doctrine updates | Applied selectively, known gaps unresolved |

**Pre-session stability check**: each body has explicit PASS criteria
(e.g. Self: internal clock −2 to 0, no active Hunter state above level 2,
prior Psyq coherence>0.70) — any FAIL routes to a **Recovery Protocol**
(Self: CW reset + 2 min physical movement + a non-judgmental FIL entry,
minimum 10-minute wait before rechecking; Market: observe-only if no
directional prior or EB₀<4; Framework: complete missing IPS layers / file
the overdue AAR before proceeding — this takes priority over market prep).

# The Daily Selection Audit (four domains — go/no-go gate)

| Domain | Fails when |
|---|---|
| Physiological | Sleep <6.5h (unilateral No-Go alone), or 2 of {hydration, nutrition, physical condition} below baseline |
| Psychological | Emotional residue above baseline, or the prior session's reset wasn't completed |
| Doctrinal | Any required documentation unfiled (AAR, FIL within 48h, Psyq log), or a recent violation's causal analysis is incomplete |
| Contextual | A hard external commitment, a scheduled macro event the Brief can't account for, or drawdown-driven size-reduction need |

| Outcome | Condition | Authorization |
|---|---|---|
| GO — full standard | All 4 pass | Standard sizing, all three layers within OPORD |
| GO — reduced standard | 3 of 4 pass | Max 2 layers, size at 65%, EB₀ effectively −3, no holds past CW+0.7 |
| NO-GO | ≥2 fail, or Physiological alone fails | Observation only — no OPORD, no positions |

**Voluntary stand-down**: even a passing DSA can be voluntarily overridden
using FIL-accumulated personal pattern data (e.g. "sessions after a
3+ win streak reliably show overconfidence-driven execution failures").
Trading through a known voluntary-stand-down condition is framed as
creating a liability to one's own framework — the direct analogy to SFAS
candidates who "ring the bell" appropriately rather than push through.

# The Battle Rhythm (three nested cycles)

- **Daily**: Pre-session (DSA → IPS → Stability Check → Mission Brief written and closed → CW pre-reading logged) → Session (ground-lead mode only, no IPS/Brief revision, abort conditions monitored continuously) → Post-session (Qu'an Log completed, Psyq log, AAR's 4 questions, FIL entry if warranted, toroidal reset executed *before* anything else) → Inter-session (no market analysis at all — structural recovery time).
- **Weekly** (fixed, Sunday, practitioner-framework interaction only — no market prep): Weekly Aggregate AAR; FIL review (tag entries `[CONFIRMED]`/`[DISCONFIRMED]`, candidates for promotion to a named IPS Layer-1 criterion); Entropy Budget calibration (compare predicted vs. actual consumption); Selection Standard review (were any No-Go sessions traded anyway?); Dossier completeness check.
- **Monthly**: the FIL rereading protocol runs here, producing a Practitioner's State Assessment (one paragraph per Three-Body domain), a Framework Calibration Report (one paragraph per IPS Layer), and a single evidence-based Development Priority for the coming month.

**The 252-session horizon** (one trading year): the minimum sample size
for the practitioner's performance tensor (regime × wave × mental state
× time-of-day) to become statistically meaningful — before that, cells
have <10 samples and are "statistically inert."

# The Practitioner's Dossier — eight documents

| Document | Function | Frequency |
|---|---|---|
| Pre-Session Intel Summary (PSIS) | Intelligence | Every session, pre-market |
| Mission Brief (OPORD) | Planning | Every trade |
| Qu'an Log | Execution record | Every trade, post-execution |
| Risq Protocol Log | Risk-dimension tracking | Every trade, during execution |
| After-Action Review | Learning | Every session, post-market |
| Field Intelligence Log (FIL) | Observation | ≥3 entries/week |
| Psyq Log | Psychological state | Every session, pre and post |
| Doctrine Log | Framework evolution record | As produced by AARs/monthly review |

These **compound** rather than merely coexist — PSIS constrains what a
Mission Brief can validly claim; the Mission Brief is the AAR's source
document for "what was planned"; individual Qu'an Log entries become the
performance tensor at scale; every AAR writes exactly one entry to the
Doctrine Log before it's applied to any operating document. **The dossier
begins on session 1**, deliberately incomplete and imprecise — its value
is the visible trajectory of improvement across the series, not any
single entry's quality.
