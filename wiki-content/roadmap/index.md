# Roadmap

Forward-looking design specs for the terminal — **not yet built** unless
a concept explicitly says otherwise. Each is grounded in an "implementation
prompt" source document and cross-checked against the current codebase to
mark what, if anything, has already shipped.

* [Bookmap research environment](bookmap-research-environment.md) - session isolation, liquidity layering, event detection, inspection tools.
* [Price tab annotation framework](price-tab-annotation-framework.md) - persistent annotations, drawing tools, multi-view rendering.
* [Rolling Analysis Engine — full vision](rolling-analysis-full-vision.md) - the objectives beyond the shipped core aggregation.
* [Chronometric Heatmap — full vision](chronometric-heatmap-full-vision.md) - multi-sheet temporal dropdown, cross-sheet correlation, publication integration.
* [Universal screenshot capture](universal-screenshot-capture.md) - a reusable "capture current visualization" service for publishing.
